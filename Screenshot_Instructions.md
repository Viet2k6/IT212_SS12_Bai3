Hướng dẫn nộp bài "Ảnh chụp màn hình quá trình ép AI sửa code/thêm tính năng":

Chào bạn, vì mình là AI nên mình không tự chụp màn hình khung chat của chúng ta được. Bạn hãy làm theo các bước sau để hoàn thiện File 03 nhé:

1. Trở lại khung chat giữa bạn và mình (Antigravity).
2. Chụp ảnh màn hình đoạn tin nhắn bạn yêu cầu mình sinh code, và đoạn tin nhắn này.
3. Nếu giảng viên yêu cầu phải có bước "ép AI sửa code", bạn hãy gõ thêm một câu lệnh yêu cầu vào khung chat. Ví dụ: "Hãy thêm validation tuổi >= 18 cho class AccountRegisterRequest". Sau đó chụp lại quá trình mình phản hồi và sửa code cho bạn.
4. Lưu các ảnh chụp màn hình này vào file word/pdf hoặc lưu dưới dạng .png/.jpg để nộp dưới tên File 03 theo yêu cầu của bài tập.

Chúc bạn hoàn thành tốt bài thi!
